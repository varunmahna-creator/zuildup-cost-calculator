# ZuildUp Cost Calculator

Interactive cost calculator for ZuildUp construction packages.

## Features

- Real-time cost calculation for Premium, Platinum, and Royal packages
- Zone-based pricing (A, B, C, D)
- Interactive toggles for various construction options
- Responsive design with ZuildUp branding

## Usage

Open `zuildup-cost-calculator.html` in any web browser.

## Deployment

This is a standalone HTML file that can be deployed anywhere - GitHub Pages, Netlify, or any web server.
